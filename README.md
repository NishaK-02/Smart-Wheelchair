## Description
This project presents a Smart Wheelchair system using an STM32 microcontroller.
The wheelchair is controlled using an analog joystick and enhanced with ultrasonic
sensors for real-time obstacle detection and automatic stopping.

## Features
- Joystick-based manual control
- Obstacle detection
- Automatic motor stop for safety
- LED and buzzer alerts
- Low-cost and scalable design

## Hardware Used
- STM32F103 Blue Pill
- L298N Motor Driver
- Ultrasonic Sensors (HC-SR04)
- Joystick Module
- DC Motors
- Buzzer and LEDs

## Software Used
- Arduino IDE
- STM32 Arduino Core
- Embedded C

## Applications
- Smart wheelchairs
- Assistive mobility systems
- Microcontroller and Embedded based project
